## ZegoLiveStreamingListener

<a id="onPKStarted"></a>

### onPKStarted

>
> This callback will be triggered  when pk battle start
>
>- function prototype:
>```java
>void onPKStarted();
>```
>- example:
>```java
>ZegoLiveStreamingManager.getInstance().addLiveStreamingListener(new ZegoLiveStreamingListener() {
>   @Override
>    public void onPKStarted() {
>       //...  
>   }
>   //...
>});
>```

<a id="onOutgoingPKBattleRequestRejected"></a>

### onOutgoingPKBattleRequestRejected

>
> This callback will be triggered when you send a pk battle request,and it was rejected by another host.
> The reason can be one of enum ZegoLiveStreamingPKBattleRejectCode.listed as HOST_REJECT, USER_NOT_HOST, IN_PK, LIVE_NOT_STARTED, ALREADY_SEND, ALREADY_RECEIVED
>
>- function prototype:
>```java
>void onOutgoingPKBattleRequestRejected(int reason, ZegoUIKitUser anotherHostUser);
>```

>- example:
>```java
>ZegoLiveStreamingManager.getInstance().addLiveStreamingListener(new ZegoLiveStreamingListener() {
>   @Override
>    public void onOutgoingPKBattleRequestRejected(int reason, ZegoUIKitUser anotherHostUser) {
>       //...  
>   }
>   //...
>});
>```
